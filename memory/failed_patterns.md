## Rules for Adding a Failure

Add a pattern only after data supports it:
- CTR below 3% after thumbnail/title rescue.
- AVD below 35% and retention drop confirms weak hook or weak list item.
- 3 consecutive videos in one pillar underperform against target.
- **NEW: Script rejected for being too similar to competitor content or following a template.**

## Current Failures

### 🔴 CRITICAL: Template-Based Script Writing
**Status:** Active risk — not yet observed in published content (channel not launched)
**Source:** User directive + YouTube policy research (July 2025 update)
**Description:** Writing scripts that follow a fixed template structure (Fear → Authority → Mechanism → Story → CTA) with repeated transitions, formulaic hooks, and interchangeable content. This is the #1 trigger for YouTube's "inauthentic content" enforcement.
**Evidence:** YouTube terminated Screen Culture and KH Studio (Dec 2025). Thousands of channels demonetized July 2025. Reddit r/PartneredYoutube reports mass enforcement against template-based AI content.
**Prevention:** 
- Read `knowledge/YOUTUBE_POLICY_CONSCIOUSNESS.md` before every script
- Apply the "Remove the Name" test
- Ensure every script has unique structure
- Include original insights not found in competitor content
- Vary transitions, pacing, and hooks across all videos
