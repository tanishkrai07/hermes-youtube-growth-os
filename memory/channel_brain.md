## ⚠️ ANTI-TEMPLATE RULES (READ BEFORE EVERY SCRIPT)

These rules exist because YouTube is actively mass-banning channels that produce repetitive, template-based AI content. Violating these rules risks demonetization or termination.

1. **NO fixed script structure.** Every script must have its own organic flow. The structure emerges from the content, not imposed on it.
2. **NO repeated transition phrases.** Vary how you move between sections every single time.
3. **NO formulaic hooks.** The cold open should be genuine and specific to THIS topic, not a recycled pattern.
4. **Varied pacing across videos.** Some fast, some slow, some punchy, some reflective. Never the same rhythm.
5. **Original insight in EVERY video.** At least one piece of information/perspective NOT found in other elder-health content.
6. **Human voice over polish.** Real person talking, not a script being performed. Varied sentence lengths. Natural flow.
7. **The "Remove the Name" test.** If you can swap any other doctor's name in and it still works, it's too generic. Rewrite.
8. **Emotional authenticity.** Don't direct emotions with stage directions. Write words that CREATE emotion naturally.
