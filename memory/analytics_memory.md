# Analytics Memory

Last updated: 2026-05-16

No channel analytics have been added yet.

## Required Dashboard Fields

For each video, capture:

- Date published
- Video title
- Pillar
- Persona
- Views
- Impressions
- CTR
- Average view duration
- Average percentage viewed
- Likes
- Comments
- Shares
- Subscribers gained
- Traffic source split
- Top geography
- Thumbnail version used
- Title version used
- First 48-hour views

## Benchmark Targets

- CTR: 4-7%
- AVD: 45-55% of video length
- Subscriber conversion: 1-3% per video
- Comments: 5-15 per 1,000 views
- First 48-hour views: 3,000+ by week 4
