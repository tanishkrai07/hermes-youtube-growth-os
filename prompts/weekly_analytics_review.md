# Weekly Analytics Review Prompt

Use this after adding weekly YouTube Studio dashboard data.

Read:

- Current context pack
- Weekly analytics export
- Winning patterns
- Failed patterns

Return:

1. Top 3 videos this week and why they won.
2. Videos below 3% CTR and exact thumbnail/title rescue action.
3. Videos below 35% AVD and likely retention cause.
4. Best pillar to double down on next week.
5. Any pivot trigger activated.
6. Which competitor pattern we should copy next.
7. Updates for `winning_patterns.md`, `failed_patterns.md`, and `next_actions.md`.

Benchmarks:

- CTR target: 4-7%
- AVD target: 45-55%
- Subscriber conversion target: 1-3%
- Comments target: 5-15 per 1,000 views
