# Self-Improvement Loop Prompt

Run once per week after analytics review.

Read:

- Current context pack
- Winning patterns
- Failed patterns
- Weekly analytics report
- Competitor memory

Answer:

1. What did Hermes predict this week?
2. What actually happened?
3. Which SOP rule worked?
4. Which SOP rule failed or needs refinement?
5. What did competitors do that we missed?
6. What should change in prompts, memory, or scoring?
7. What should remain unchanged?

Output:

- Proposed memory updates
- Proposed prompt updates
- Proposed next-week strategy

Rules:

- Do not rewrite raw SOPs.
- Do not overfit to one video unless performance is extreme.
- Preserve exact evidence: titles, CTR, AVD, views, dates.
