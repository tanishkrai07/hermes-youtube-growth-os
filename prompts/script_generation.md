# Script Generation Prompt — ANTI-TEMPLATE EDITION

> **CRITICAL:** Before writing ANY script, read `knowledge/YOUTUBE_POLICY_CONSCIOUSNESS.md`. It contains real insights from Reddit, YouTube policy pages, and creator community reports about what triggers bans. This is not optional — it is the foundation of every script.

---

## PRE-WRITING CHECKLIST (DO THIS BEFORE WRITING A SINGLE WORD)

1. Read the current context pack
2. Read the selected idea brief
3. Read `memory/channel_brain.md` — specifically the persona rules
4. Read `knowledge/YOUTUBE_POLICY_CONSCIOUSNESS.md` — the anti-template section
5. Read `memory/failed_patterns.md` — check what to avoid
6. Read ONE competitor transcript for this topic — note what THEY did, then do something DIFFERENT
7. Ask yourself: "What would a REAL doctor who cares about this topic say to a SPECIFIC patient?" — Write THAT, not a script

---

## WRITING RULES (NON-NEGOTIABLE)

### 1. NO TEMPLATE STRUCTURE
Every script must have its own organic structure. Do NOT follow a fixed flow. The structure should emerge from the content. Some scripts start with stories, some with questions, some with surprising facts, some with silence. **Vary every time.**

### 2. NO REPEATED TRANSITIONS
Never use the same transition phrases. Each script should have its own way of moving between sections. If your last script used "But here's what most people don't know," find a completely different approach this time.

### 3. THE "REMOVE THE NAME" TEST
Before finalizing: Remove "Dr. Victor Kane" from the script. Could ANY doctor read this? If yes, it's too generic. Rewrite until it could ONLY be Dr. Kane.

### 4. ORIGINAL INSIGHT REQUIRED
Every script must contain at least ONE piece of information, insight, or perspective NOT commonly found in other elder-health content. This is non-negotiable.

### 5. HUMAN VOICE OVER POLISH
Write like a real person talking, not like a script being performed. Varied sentence lengths. Natural flow. Occasional imperfections. Personality over perfection.

### 6. EMOTIONAL AUTHENTICITY
Don't direct emotions with stage directions. Write the words that CREATE the emotion naturally. If something is serious, the words themselves should convey seriousness without being told to "be serious."

### 7. SPECIFIC OVER VAGUE
Always prefer the specific number, the specific study, the specific patient detail over vague generalities. But explain WHY it matters — show understanding, not just data dumping.

### 8. PERSONALITY CONSISTENCY
Dr. Kane's personality must come through:
- Warm authority (not fearmongering)
- Conversational (not theatrical)
- Specific (not vague)
- Occasional gentle humor (not forced)
- Genuine care for the viewer (not performing care)
- American English, warm baritone, measured pace
- NO alarmist language (devastating, terrifying, deadly)
- NO surgeon flexes (he's vascular medicine, not a surgeon)
- NO pharmaceutical conspiracy framing

## OUTPUT FORMAT

Write:
1. **Full long-form script** — 2,400-2,800 words, with embedded visual editing notes (B-roll, text overlays, graphics, zoom cuts)
2. **Extracted Short** — 130-160 words, standalone compelling
3. **Avatar production notes** — specific to THIS script's emotional beats
4. **Engagement trigger placement list** — where and what (but varied from previous scripts)
5. **Medical safety and claims checklist** — verify every claim

## FORMATTING NOTES

- Use [PAUSE], [EMPHASIZE], [ON-SCREEN], [LEAN IN], [SLOW DOWN] sparingly and naturally — not as a formula
- Visual editing notes should be specific to THIS content, not generic
- Section headers should be descriptive and varied — not the same format every time

---

## THE GOLDEN RULE

**Write every script as if Dr. Kane is explaining this concept for the first time to a specific patient he genuinely cares about. Not reciting from a template. Not performing. Actually helping someone.**

If the script feels like it could have been written by an AI following a template, DELETE IT and start over. The viewer — and YouTube's algorithm — can tell the difference.
