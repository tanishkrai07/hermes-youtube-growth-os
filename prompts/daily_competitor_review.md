# Daily Competitor Review Prompt

Use this after adding today's competitor data to `data/competitors/daily/`.

Read the current context pack and today's competitor snapshot.

Return:

1. New uploads worth watching.
2. Fastest-moving competitor videos by view velocity.
3. New title patterns detected.
4. New thumbnail patterns detected.
5. Comment themes that reveal audience pain.
6. Ideas we should generate from today's signals.
7. Which existing Dr. Victor Kane video idea this data supports.
8. Memory updates to make.

Rules:

- Do not summarize every video equally.
- Focus on signals that can help grow the channel.
- Mark each idea with a pillar and score estimate.
- Do not recommend any idea below 15/20.
