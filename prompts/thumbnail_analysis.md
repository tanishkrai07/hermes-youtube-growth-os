# Thumbnail Analysis Prompt

Use this when adding a competitor thumbnail or designing Dr. Victor Kane thumbnails.

Analyze:

1. Thumbnail text.
2. Background color.
3. Doctor position.
4. Doctor expression.
5. Prop or visual trigger.
6. Layout type.
7. Emotional trigger.
8. Mobile readability.
9. What to copy.
10. What to avoid.

Then create:

- Thumbnail Version A: red/yellow fear.
- Thumbnail Version B: dark/navy authority.
- A/B test variable.
- File naming recommendation.

Rules:

- Text must be ALL CAPS.
- Max 6 words primary text.
- No text identical to title.
- Only one main prop.
- Doctor face must remain clear.
