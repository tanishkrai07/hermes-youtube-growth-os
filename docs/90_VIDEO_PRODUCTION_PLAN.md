## Quality Control Process

For each script:
1. **Write** — Follow anti-template rules
2. **Analyze** — Check for template patterns, competitor similarity
3. **Fix** — Rewrite any sections that feel generic or templated
4. **Verify** — Pass the "Remove the Name" test
5. **Approve** — Mark as production-ready
